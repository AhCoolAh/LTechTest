# LTechTest

## Getting Started

```
$ git clone https://github.com/AhCoolAh/LTechTest.git
$ cd LTechTest
$ sudo gem install cocoapods
$ pod install
$ open LTechTest.xcworkspace
```

## Run parameters

* Target - LtechTest
* iOS Deployment Target - 15.2
* Swift Language Version - Swift 5
